DIMARE
=============

Instalação
-----------

```
composer install
npm install
```

## Copiar conteúdo de .env.example
## Criar novo .env e colar o conteúdo

## Gerar nova key para o .env
```
php artisan key:generate
```

Uso
-----------

## Subir o Laravel
```
php artisan serve
```

Durante o desenvolvimento, para recompilar o stylus
```
npm run watch
```
